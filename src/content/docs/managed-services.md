## Sparkable Managed OpenEMR

[Sparkable](https://sparkable.cc) provides fully managed OpenEMR hosting with the REST API and FHIR API pre-configured. No need to worry about Docker setup, SSL certificates, or API configuration.

## Features

| Feature | Description |
|---------|-------------|
| **Pre-configured API** | REST and FHIR APIs enabled and optimized out of the box |
| **Auto-scaling** | Infrastructure scales with your practice size |
| **SSL/TLS** | Automatic HTTPS certificate management |
| **Backups** | Daily automated backups with point-in-time recovery |
| **Monitoring** | 24/7 uptime monitoring and alerting |
| **Updates** | Managed OpenEMR updates and security patches |
| **CORS** | Cross-Origin Resource Sharing pre-configured |
| **Support** | Dedicated technical support for API integration |
| **FHIR Compliance** | Validated FHIR R4 compliance |
| **Custom Domain** | Use your own domain with the API |

## Getting Started

1. Visit [sparkable.cc](https://sparkable.cc)
